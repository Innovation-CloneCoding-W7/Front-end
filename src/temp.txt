# 상품 카테고리.
charging - 충전 용품
vehicle-accessories - 차량 액세서리
apparel - 의류
lifestyle - 라이프스타일
service-installables - 서비스 설치 용품

GET - /category/{category_name} => 충전 용품에 해당하는 상품 모두 불러오기.
예시) /category/charging

# 상품 상세 페이지.
GET - /product/{product_name} => 해당 상품 이름에 해당하는 상세 정보 불러오기.
예시) /product/wall-connector-pedestal
